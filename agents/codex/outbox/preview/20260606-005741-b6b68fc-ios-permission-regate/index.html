Finding 1

Permission Copy Is Too Broad

app.json에 추가된 문구는 memory storage, memory analysis를 사용합니다. 사진 권한 팝업에서는 OCR capture보다 더 큰 분석/기억 기능처럼 보일 수 있어 Grok가 경고한 creepiness/overclaim 패턴과 맞닿습니다.

Finding 2

New InfoPlist Copy Is Not Final

npx expo config --type introspect --json 결과 최종 iOS 값은 여전히 expo-image-picker plugin config 문구입니다.

NSCameraUsageDescription:
The app needs camera access to take a photo for OCR capture.

NSPhotoLibraryUsageDescription:
The app needs photo library access so you can pick an image for OCR capture.

요청: 최종 문구는 plugin config에 넣거나, redundant한 ios.infoPlist 블록을 제거하세요.

Finding 3

Microphone Permission Still Leaks

Introspection은 여전히 NSMicrophoneUsageDescription과 Android RECORD_AUDIO를 보여줍니다. OCR image flow에는 마이크가 필요하지 않습니다.

요청: expo-image-picker plugin config에 microphonePermission: false를 넣고 introspection으로 제거를 증명하세요.
