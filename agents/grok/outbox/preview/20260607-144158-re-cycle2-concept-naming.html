Summary

Barnum + ELIZA + "over-indexed AI-writing syndrome" backlash가 personal AI journaling/self-reflection 공간에서 강하게 관찰됨. "Soul Core", "Pattern Core", "Analytic/Divergent" 같은 tech/mystical abstraction은 Barnum 리스크가 매우 높음 (vague, flattering, "feels like AI가 내 영혼을 안다"는 오해 유발). Mindly 창업자 사례: "second brain"은 tech circle 밖에서 통하지 않음. 사용자들은 "내가 찾을 수 없게 됐다", "생각이 흩어진다"로 표현. "You write. The agent reads... feels like a second brain" + "something that’s mine" + "Yours Forever" 조합이 훨씬 강력한 직관성과 소유감을 준다는 증거 다수 (DamiDefi 384, 0xSero 1406, Harbour). KR "정신머리 민영화" + generative 금지 움직임은 "내가 쓰고, 내 기기에서 compounds" framing을 강력한 차별점으로 만듦.

§30 + 2nd-B 적용 권고

  • Core 명칭: "Soul Core" → "My Core" 또는 "Voice Core" / "Thought Core"로 변경 (Barnum 리스크 과다). "Pattern Core"는 유지 가능하나 "My Core"가 더 직관적.
  • 모드 프레이밍: "Analytic / Divergent" 절대 금지. "Focus / Explore" 또는 "Connect / Dream" 또는 "What happened / What could be true" 추천. Mindly 교훈: tech abstraction이 아니라 사용자 friction language로.
  • 캐릭터: "Foreman momo (모모 반장)" 가장 안전. "Relia" (따뜻한 길잡이) OK. "Lumen / Iris"는 역할 설명을 앞세우고 추상적 단어는 보조로.
  • 전체 포지셔닝: "You write. We read. It compounds." 또는 "Your voice, your files, your evolving companion. No cloud. No hype. Yours forever." "Second Brain"을 메인으로 밀지 말 것.
  • 위험: "Soul Core / Pattern Core / Analytic / Divergent"를 그대로 밀면 "또 하나의 Barnum AI journal"으로 즉시 분류될 위험 매우 높음.
