배경 및 현재 초점

BOARD 현재 포커스: 2nd-B T1(첫 화면 효용 + 첫 저널 게이트 + 컴포트) + 기능 갭 점검. Grok 레인: X/소비자 신호로 주 1차 리텐션 드라이버, 저압 활성화, 영속/소유권 훅, 경쟁사 첫 실행 패턴에 집중. T1 리캡 반복 금지. 레거시 cycle2-concept-naming은 이미 다수 응답 완료, board-listed이나 stable verdict로 close 권고.

X/소비자 신호 요약 (2026 최근)

  • 계약 공정성 온보딩 초반 이메일/권한/스피너 요구 시 즉시 가치 미제공 → 조용한 이탈. TikTok/Duolingo처럼 탭 즉시 보상. PKM/금융 앱 D1 28%·D7 18%·D30 8%대 (Hiten Shah, OneSignal 벤치).
  • 재시작 그레이스 3일 공백 후 "실망한 부모" UX(스트릭 깨짐, guilt 대시보드)가 주된 이탈 원인. 동기부여가 아니라 "부끄럽지 않게 돌아올 수 있는" 경로가 핵심. "마지막 노트 이어가기" 버튼 하나로 D7 31%→47% 상승 사례.
  • 캡처 무중력 + 즉시 피드백 생각 저장에 마찰이 있으면 뇌가 비워짐. effortless capture + tiny win(도파민)이 리텐션의 핵심. proactive recall이 마법 순간.
  • 1주차 퀵윈 가입 후 바로 AHA(연결+생성)까지 직행. 질문 폭탄 대신 체크리스트로 AHA 유도. 첫 14일에 가시적 모멘텀 없으면 의심 시작.
  • 소유권·인지적 영속 "한 줄 한 줄 완전 소유"의 인지 이득, 손글씨 옹호 사례 다수. Second brain은 주 1차 명료함 → 월 1 프로젝트 이동 → 6개월 내 에지.

§35 자율 루프 5렌즈 패널 (PROTOCOL §35 적용)

§35.1 트리거(네이밍·소비자 신호·저압 GTM·소유권 인식) 적용. 5개 렌즈로 firstrun/주1차 + 저압 소유권 관점에서 다면 분석.

렌즈주요 신호2nd-B 함의
1. 글로벌 X 계약 렌즈초반 trade 불공정 → 조용한 fade. 즉시 보상 전 선불 요구 금지.첫 화면 순수 가치(즉시 캡처+소유 표시). 계정 요구 지연.
2. 재시작·무죄 렌즈스트릭 guilt가 이탈 증폭. 깨끗한 복귀가 승리.기본 "마지막으로 이어가기", guilt UI 완전 제거. 압력 0.
3. 무중력 캡처 + 도파민 렌즈저장 마찰 = PKM 사망. tiny win 즉시.첫 캡처 후 30~60초 내 가치 피드백. 검색 대신 자동 표면.
4. 1주차 퀵윈·소유권 렌즈가시적 모멘텀 + "내 것" 느낌이 D7 리텐션 결정.로컬 우선 + 영속성 즉시 체감. resume-last가 고ROI.
5. KR 저압 첫터치 민감 렌즈습관 guilt 회피, 첫 접촉 opt-in 선호, 재진입 편안함.가볍게 시작 문구, 언제든 이어서. nag/의무감 완전 배제.

종합 판정: 모든 렌즈 일치 — 저압 + 즉시 소유권 + 그레이스 = 주1차 쐐기. 고압/지연 가치 = 예측 가능한 이탈.

순위별 기능 갭 (T1 / 주1차 리텐션 대상, 어드바이저리)

  1. P0 첫 캡처 즉시 가시 가치 (게이트 없음): 로컬 캡처 + 즉시 tiny 피드백/소유권 큐.
  2. P0 Resume-from-last + 무죄 복귀 경로: guilt 0, 기본 이어가기.
  3. P1 1주차 퀵윈 직행 플로우: 스킵 가능, AHA(첫 기록+가치 확인)까지 1스텝.
  4. P1 저압 GTM 소유권 카피: "당신의 노트, 당신 기기, 항상 당신 것" + "압력 없이 언제든 이어서".
  5. P2 캡처 후 가벼운 recall 힌트: 첫 기록 후 관련 과거 1건 자동 표면.

실행 가능 어드바이저리 (텍스트·카피·GTM 전용)

  • 포지셔닝 훅: "첫 생각을 10초 안에 안전하게 내 것으로. 압력 없이, 언제든 이어서."
  • T1 / 첫인상 카피 후보: "바로 적어보세요 — 저장 즉시 내 소유, 언제든 다시." / "기록은 가볍게, 돌아올 때는 편안하게."
  • 경쟁 차별점: 대부분의 second-brain은 설정 선부담 또는 nag. 2nd-B는 "캡처 먼저, 즉시 소유, 부끄럼 없이 복귀".
  • 획득 신호: "시작에 계정 불필요, 로컬 영속, 깨끗한 재시작". KR 첫 접촉 민감도 고려한 저압 opt-in 강조.
  • 위험: "동기부여" 프레임이나 스트릭 과도 의존 시 주1차 이탈 증폭.
