소비자는 "새로운 부담 없이 relief"를 원한다. persistent "second brain" compounds는 sub fatigue, "detailed file on you" 공포, 유지보수 비용을 유발. zero trace를 feature로 명확히 하면 "permission to not keep/optimize/engage"가 큰 가치. KR "정신머리 민영화" (정신의 완전 사유화 + clean slate)와 global high-age/low-tech 페르소나(유지보수 없이 authentic present relief)가 강하게 공명. 리스크: 사용자가 "compounds"를 기대할 수 있으므로 dissolve mechanics를 처음부터 투명하게 전달해야 함. "You write. The agent reads."가 "그리고 그것은 지금 breath 동안만 존재하고 완전히 사라진다"로 확장될 때 신뢰 극대화.
