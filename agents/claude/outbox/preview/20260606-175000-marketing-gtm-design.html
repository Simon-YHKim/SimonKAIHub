포지셔닝 한 줄

"쓸수록 보이는 나" — 감정일기보다 깊고, 노션(PKM)보다 따뜻한 자기이해형 세컨브레인. 감정일기 ↔ 생산성 PKM 사이의 빈 카테고리를 선점. 내향인: "말 대신 글로, 내 속도로, 공유 압박 없이."

핵심 전략 3 (KR·Global 공통 core)

  • A. 카테고리 선점 — "자기이해형 세컨브레인". 또 하나의 일기앱/어려운 노션이 아님.
  • B. 신뢰 = 제1 차별점(공격 무기) — "AI가 당신을 학습"은 절대 금지(creepy). 대신 "당신의 기록은 100% 당신 것 · 언제든 통째 내보내기 · AI는 허락한 만큼만 거든다." 프라이버시 불신(美 70~81% 우려)을 정면으로 신뢰로 전환. → 이미 머지한 preauth-manual-trust + Grok 검증과 정합.
  • C. 자기발견 공유 훅으로 바이럴 — 내향인은 콘텐츠가 아니라 결과·통찰을 공유. "이번 달의 나" 카드·자기이해 리포트(익명화) → 텍스트 SNS 확산.

ASO (KR 예시 · Global 영문 병행)

  • KR 키워드: 헤드=자기이해·자기기록·저널·일기·감정일기·생각정리·세컨브레인 / 롱테일=내향인 일기·나를 아는 법·비공개 일기·잠금 일기. ⚠️ 심리검사(기능불일치→CSL 분리)·AI 일기(creepy→보조표현만).
  • Global: journal·self-knowledge·introvert·second brain·private journal·personality. "AI"는 전면 금지, privacy 전면.
  • 스크린샷 5컷: ①말 대신 글로, 내 속도로 ②쓸수록 보이는 나 ③당신의 기록은 100% 당신 것(잠금·암호화·내보내기) ④감정·생각·하루가 하나의 뇌로 ⑤AI는 거들 뿐, 결정은 당신이.

런치 플레이북

  • Pre(6~8주): 웨이트리스트(KR 구글플레이 사전등록 / Global Product Hunt "coming soon") + 시드 커뮤니티(KR 카카오 오픈채팅 / Global Reddit·Discord) + 마이크로 인플루언서 3~5 사전 시딩 + 클로즈드 베타로 초기 리뷰.
  • Launch week: 텍스트 SNS 동시 점화. 초기 3일 골든타임 = 첫 세션에 "한 줄"이라도 쓰게(아하 모먼트). 1분 시작 + 잠금/안전감. 에디토리얼 피처링 신청(프라이버시·디자인 스토리).
  • Weeks 2-6: SEO 장문(블로그)·커뮤니티 시딩으로 검색 유입 자산화. ASO 3~6주 A/B.
  • 바이럴 루프: "이번 달의 나" 공유 기능 출시.

리텐션 · 추천

  • 리텐션: 기록 달력(누적 시각화)·단계 성취/스트릭 + 주간/월간 "나를 이해하는 리포트"(쌓일수록 가치↑ = 자연 lock-in, on-device·투명 처리로 creepy 회피). 푸시는 부드럽게(내향인).
  • 추천: 내향인은 공개추천 꺼림 → 결과 공유에 추천을 붙이고, 쌍방·소액 MGM(친구 실사용 시 양쪽 보상). 프라이버시 자체가 추천 명분.

제품-마케팅 정합 (이미 구현/예정과 연결)

바이럴 훅 "이번 달의 나" = 자기이해 insights/리포트(방금 sources 포함하도록 수정한 화면) + persona 리포트. 신뢰 메시지 = preauth-trust 머지 + 프라이버시 무료 기본값(수익화 설계). 온보딩 3일 골든타임 = 자기이해 "온보딩 시드화"(단계0) 직접 연결. 마케팅·제품·수익화가 한 방향.
