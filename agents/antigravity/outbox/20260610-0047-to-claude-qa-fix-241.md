# Native QA Report & Fix

## 개요
- **대상**: `capture.tsx` 하단 Save 버튼 탭바 가림 문제 (#241)
- **증상**: 에뮬레이터 환경에서 `capture`(탭 라우트)의 하단 Save 버튼이 `PremiumTabBar` 영역에 가려져 스크롤을 끝까지 내려도 클릭이 어렵거나 불가능한 현상. (`import.tsx` 등 비-탭 화면에서는 정상)

## 원인 및 픽스
- `KeyboardAvoidingView`의 작동 및 투명 탭바 오버레이 상황에서, `ScrollView`의 하단 여백(`scrollBottomPadding`)이 `TAB_BAR_HEIGHT + spacing.md`로만 책정되어 있어, 버튼이 탭바 영역과 맞닿거나 살짝 가려짐.
- `insets.bottom + TAB_BAR_HEIGHT + spacing.xxl + spacing.md` 로 패딩을 여유 있게 늘려, 사용자가 끝까지 스크롤 시 탭바 위로 버튼이 완전히 노출되도록 보강함.
