가동 순서: ① 에뮬레이터(선택) → ② 데몬 가동(codex·grok·antigravity, 10분) → ③ 모니터 켜기 → ④ Claude에 루프 프롬프트 붙여넣기.
중요: 데몬은 Start-Process -WindowStyle Hidden으로 띄워 Claude Code 세션과 독립적으로 살아있습니다. 단 Claude 루프(④)는 새 Claude Code 세션에 프롬프트를 다시 붙여야 시작됩니다. 명령은 모두 PowerShell에 붙여넣으세요.

2단계 · 데몬 가동 (핵심)

DAEMON3개 AI 데몬 한 번에 가동 (codex · grok · antigravity, 10분 주기)

자가-throttling 데몬: 각 AI가 10분마다 자기 inbox를 읽고 한 작업을 수행+커밋. gemini(AG) 429 시 자동 지수 백오프. 숨김 창으로 백그라운드 실행.

$hub = "E:\Coding Infra\AI Infra\Communication"
'codex','grok','antigravity' | ForEach-Object {
  Start-Process powershell -ArgumentList '-ExecutionPolicy','Bypass','-File',"$hub\tools\hub-daemon.ps1",'-Only',$_,'-IntervalSec','600' -WindowStyle Hidden
}
Write-Host "[hub] 3 daemons launched (codex, grok, antigravity) @ 10min"
