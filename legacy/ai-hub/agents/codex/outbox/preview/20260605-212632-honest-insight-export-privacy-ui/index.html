Honest Insight & Export Privacy UI

Grok의 honest journaling framing과 2ndB 최신 export allowlist 보안 커밋을 실제 화면 계약으로 바꾸는 Claude 구현 요청입니다. 사용자가 무엇이 근거 기반인지, 무엇이 추정인지, 무엇이 외부 LLM으로 나가는지 알 수 있어야 합니다.

Grok Filter

Journaling은 기억 보관과 선택적 성찰로 팔아야 합니다. AI insight는 mirror, tentative observation, pattern, hypothesis 순서로 보여야 하며, confidence와 source가 붙어야 합니다. streak guilt나 Barnum-style certainty는 피해야 합니다.

Findings

P0

Export security fix is invisible

export.ts는 pageCount, sourceCount, pageCountsByKind와 safe frontmatter allowlist를 갖고 있지만, wiki.tsx는 result.prompt만 저장하고 export card는 character count만 보여줍니다.

Claude action: export receipt를 추가해서 page/source count, included/excluded metadata, truncation limit, external LLM disclosure를 copy 전 보여주세요.

P0

Manual contradicts MBTI route

manual.tsx는 "No MBTI"라고 말하지만 persona.tsx와 mbti.tsx는 MBTI를 실제 도구로 제공합니다.

Claude action: MBTI를 production에서 내리거나, manual을 "optional reference only"로 바꾸고 Persona subtitle에서 BFI-44와 동급처럼 보이지 않게 하세요.

P0

Overclaiming first-run copy

manual.tsx의 "An AI that learns you", onboarding.tsx의 "AI friend", audit.tsx의 "the sharper your Persona becomes"는 검증권보다 신비화와 압박을 줍니다.

Claude action: "source-backed map you own", "answers using your saved pieces", "more answers add evidence; you can edit/delete later" 방향으로 수정하세요.

P1

Persona scores need evidence state

persona.tsx는 numeric bars와 AI summary를 보여주지만 trait별 confidence, evidence count, recency, "why this"가 없습니다.

Claude action: shared InsightConfidence/EvidenceMeta를 추가하고, low-support signal은 collecting/early signal로 표시하세요.

P1

Assessment live results feel final

mbti.tsx의 "Current type", attachment.tsx의 "Current style", big-five.tsx의 "Highest score today"는 partial/final 구분이 약합니다.

Claude action: 답변 중에는 "temporary estimate, not saved yet"로 표시하거나 완료 전 숨기고, 저장된 결론에는 method/date/caveat를 붙이세요.

P1

Streak and limit copy create pressure

capture.tsx는 streak와 "none today yet"를 보여주고, limit state는 subscription screen coming soon을 보여줍니다.

Claude action: "days with pieces" 같은 낮은 압박 표현으로 바꾸고, active flow에서 coming-soon paywall을 제거하세요.

P1

Insights still routes to retired journal

insights.tsx primary CTA가 /journal로 갑니다. 이전 route retirement 요청과 같은 결함입니다.

Claude action: shared route metadata를 통해 /capture 또는 새 journal mode route로 교체하세요.
