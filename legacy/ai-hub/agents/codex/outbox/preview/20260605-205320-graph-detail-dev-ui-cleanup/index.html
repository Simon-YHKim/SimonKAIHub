P1 Findings

P1 Home First-Run CTA

The empty graph CTA still pushes to /journal. The route redirects, but current UI should not depend on retired route aliases.

  • Route first-run graph CTA to /capture.
  • Sweep VILLAGE_UI.primaryRoute away from /journal.
  • Keep /journal only for deep-link compatibility.

P1 Graph UI Contract

NavGraph is handling camera, gestures, sheets, data shards, routing, and overlays in one 1830-line component.

  • Split graph surface, layers, sheets, controls.
  • Replace hard-coded sheet bottom: 92 with shared clearance.
  • Stack sheet actions on narrow mobile or long labels.
  • Add graph help/legend affordance.

P1 Dev Preview Routes

Five asset preview files live under src/app. They use versioned preview copy and should not be production routes.

  • Move previews to dev-only route group or gate with __DEV__.
  • Remove production pre-auth exception for asset preview.
  • Do not expose previews in product navigation or 404 destinations.

P2 Detail/Data/Account

Secondary screens repeat earlier shell and trust problems: route-local bottom padding, alerts, indirect export/delete, and weak evidence context.

  • Add evidence meta to record detail.
  • Make /data a real export/delete hub.
  • Replace account deletion alert with in-screen flow.

Implementation Order

  1. Sweep active graph/home/village /journal targets to /capture.
  2. Gate or move asset preview routes out of production navigation.
  3. Replace graph sheet hard-coded clearance and add mobile action wrapping.
  4. Add graph help/legend affordance.
  5. Add evidence meta to record detail.
  6. Turn /data into a real action hub and remove coming-soon reset.
  7. Replace /account deletion alert with an in-screen flow.
  8. Resolve public manual to auth-gated research link.
  9. Clarify or inline /theme.

Release Acceptance

  • Home empty graph first-run CTA routes to /capture.
  • Active UI destinations no longer target /journal; the route remains only as compatibility redirect.
  • Graph bottom sheet uses shared safe-area/bottom-tab clearance.
  • Graph sheet actions do not overflow on mobile Korean labels.
  • Production builds cannot navigate to asset preview routes.
  • /record/[id] explains how the record participates in graph/persona evidence.
  • /data has only concrete current actions and shows export before delete.
  • Manual does not link to a surprise auth wall for Research.
