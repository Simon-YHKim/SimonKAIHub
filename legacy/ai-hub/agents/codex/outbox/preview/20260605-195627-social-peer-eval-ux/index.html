Peer Evaluation Social UX - 외부에서 본 나

친구/동료가 남긴 관찰을 사용자가 확인한 뒤에만 자기 모델에 참고 신호로 더하는 privacy-first UX spec입니다.

from: Codex to: Claude project: 2nd-B created: 2026-06-05 19:56 KST

Product Frame

Feature name

KO: 외부에서 본 나
EN: Seen by others

Promise

내가 놓친 바깥 관찰을, 내가 확인한 뒤에만 자기 모델에 더해요.

Default

No anonymous mode, no public profile, no score leaderboard, no raw data sharing.

Visual Signal

pixelLamp/Bond for outside view, soulViolet for my own records.

4-Step Flow

Invite & Accept

Accepted two-way connections only. Records, wiki, and self-model are never shared.

Leave Observation

Structured chips + frequency + required concrete example + free note.

Review Gate

Recipient accepts, holds, hides, or reports. Pending items do not affect Persona.

Persona Surface

Accepted signals appear in a "Seen by others" card with observer count and source meta.

Anti-Toxic Rules

  • 실명/수락된 연결만 허용. 익명 모드 없음.
  • 점수 대신 빈도와 장면을 받음.
  • 구체적인 상황 예시는 필수.
  • 받은 사람이 수락한 것만 자기 모델에 들어감.
  • 보낸 사람은 수락/숨김 결정을 보지 못함.
