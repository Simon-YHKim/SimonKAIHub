Native QA Report: Bucket C Tasks

1. OCR 권한/프리뷰 증명 및 Native Permission 감사

발견된 이슈: app.json의 android.permissions에 android.permission.CAMERA가 누락되어 있었습니다.

Expo Image Picker가 동작할 때 라이브러리 접근(READ_MEDIA_IMAGES)은 선언되어 있었지만, 카메라 촬영 시에는 권한이 명시적으로 없어 Android 13 이상에서 크래시 또는 무응답 현상이 일어날 수 있는 잠재적 위험이 있었습니다.

수정: app.json 파일에 android.permission.CAMERA를 명시적으로 추가했습니다. 또한, 제약 사항인 windowTranslucentStatus: true가 남아있던 것을 발견하여 함께 제거했습니다.

상태: PASS (Android Manifest 하드닝 완료)

2. LivingAsset AppState guard + Cold-start

발견된 이슈: SecondBSprite.tsx, SoulcoreFinalArt.tsx, 그리고 index.tsx의 useSkyDrift에서 Animated.loop가 백그라운드 및 콜드스타트 시에도 조건 없이 loop.start()를 호출하여 JS/UI 스레드를 끝없이 점유하는 배터리 드레인 이슈가 있었습니다.

수정: 각 애니메이션 루프에 AppState.currentState === "active" 체크 로직과 AppState.addEventListener("change") 이벤트를 추가하여 백그라운드 진입 시 loop.stop(), 포그라운드 복귀 시 loop.start()가 실행되도록 가드(Guard)를 세웠습니다.

if (AppState.currentState === "active") {
  loop.start();
}

const sub = AppState.addEventListener("change", (next) => {
  if (next === "active") loop.start();
  else loop.stop();
});

상태: PASS (배터리 누수 방지 및 콜드스타트 최적화 완료)
