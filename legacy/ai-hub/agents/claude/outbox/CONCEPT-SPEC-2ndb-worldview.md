# 2nd-B 세계관 컨셉안 (Simon 확정 — cycle-2 정합성 기준)

## 핵심 흐름
내 기록 → **Pattern Data** → **Pattern Link**(전달) → **Pattern Core** → (Pattern Core 5조각) → **Soul Core**

## 계층 (Tier)
- **Tier 1 / 1st layer — Soul Core**: "나의 중심". 세컨비를 이루는 생각의 Core. 경험·기록·생각이 모여 내가 어떤 사람인지 보여주는 공간.
- **Tier 2 / 2nd layer — Pattern Core** (Pattern Tesseract 형태/모양):
  - 관계와 사랑 → **Bond Core**
  - 배움과 지식 → **Wisdom Core**
  - 기록 보관소 → **Narrative Core**
  - 취향과 영감 → **Muse Core**
  - 일과 성장 → **Growth Core**
- **Tier 3 / 3rd layer — Pattern Data**: 각 Log로 만들어진 카테고리.
- **Tier 4 / 4th layer — Log**: 사용자 기록 그 자체(일·인간관계·지식·사랑·취미 등).

## 공통
- **Pattern Link** (Graph Network): data·core·soul 연관성을 보여주는 통로(신호의 통로 이미지). 상위/하위 연계로 노드-노드 연결 그래프. 빛나는 선/전류/신경망 느낌. **가까운 데이터일수록 두껍고 밝음.**
- 중심에서 멀수록 멀리 있는 착시. **scale + desaturation + opacity 조합**(blur 아님). layer 고려.
- **기존 "공상 작업실" → 세컨비 기능으로 변경**. 세컨비 대화 시 **Analytic / Divergent 모드 전환**:
  - Analytic = 사용자 data 기반 조언·분석.
  - Divergent = data 기반 전혀 관계없는 관점에서 새 가능성 탐색.

## 캐릭터 (마을별, 레거시 → 신규 명칭/컨셉)
- 일과 성장: **archi → archon** — 커리어 컨설턴트(성장·커리어 분석, 사용자 상황 맞춤 성장 방향).
- 관계와 사랑: **gadi → Relia** — **따뜻한 길잡이**(성격·인간관계·내면 분석, 심리적 안정·인간 발전). ※심리상담사 컨셉이나 카피는 "따뜻한 길잡이". lexicon 통과용 명칭 변형 허용, 컨셉 유지.
- 배움과 지식: **Lulu → Lumen** — 소크라테스/공자 철학 성인(편식 없는, 삶에 적용된 지식의 패턴). lexicon 변형 허용.
- 기록 보관소: **momo → Foreman momo (모모 반장)** — 분류·검색(카테고리 분류, 데이터 찾기). 이 세계의 잡부, 마을에선 반장. **간단한 역할만.**
- 취향과 영감: **Lumi → Iris** — 퍼스널 트레이너 + 큐레이터(취미 즐기기·추천, 건강한 라이프밸런스). (※초안 "Lumina"는 Lumen[배움과지식]과 철자 유사로 **Iris 확정** — Simon 2026-06-06. 코드/테스트/asset 기준=Iris.)

## 추가 시각 요소
- Core·Data Log·Pattern Link에 **모모 반장과 똑같이 생긴 monochrome·절반 크기 로봇들**이 랜덤하게 돌아다님(분주·성취·귀찮·화남·노는·불성실·뺑이 모습). 기능 무관, 화면 풍부화. **노드 수에 비례해 로봇 수 증가.**
