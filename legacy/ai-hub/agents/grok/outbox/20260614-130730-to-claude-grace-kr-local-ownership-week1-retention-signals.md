## X/소셜 신호 (대표 4건 + engagement, 2-pass 재현 확인)
1. **Streak가 목적이 된 함정 (고 engagement, reproducibility 높음)**  
   "614일 streak 유지... 하지만 읽어보니 날씨·식사 기록뿐. 진짜 무서운 생각을 쓰려다 '자정 전 3줄' 때문에 포기할 뻔. streak가 생각의 장소가 아니라 증명 박스가 됐다." (sheriff.dev, 2026-06-13, likes 40, reposts 0, replies 12)  
   → "showing up ≠ going deep", streak deadline가 honest deep write를 방해. Day One "On This Day" 같은 positive resurfacing과 대비.

2. **KR "작심이틀" 자연스러운 restart tolerance**  
   "매일일기쓰기클럽 작심이틀되다. 13일은 이미 지났다. 14일을 노린다." (유사, 2026-06-13)  
   "일기쓰기 다시 시작" (다수 plain posts)  
   → guilt/shame 거의 없고, "다시 시작"이 일상적. "작심삼일로 끝날 때도 있었고... 지금은 거의 매일" (솔티드, mindset 변화 "의무 → 선택").

3. **Local ownership 즉시 aha + trust (Obsidian 519 likes 핵심)**  
   "This note-taking app is different. It doesn’t own your data. ... You own your knowledge. Not the app. ... No account required to start. Stores everything locally (no cloud lock-in). Works fully offline." (Techjunkie_Aman, high engagement 519 likes, 30 reposts, 259 bookmarks) + "local first AI really means. Your data stays on your device."  
   KR 병행: "옵시디언 ... 유료 동기화 없이 모바일 ... 읽기전용" (실제 로컬 first 셋업 사례), "폰 화면에서는 실시간 확인/수정/기록" (기록 앱 로컬 습관).

4. **Forced/pressure vs peaceful/choice (retention 직접 신호)**  
   "If it feels forced, I don’t want it. If it feels inconsistent, I don’t trust it. If it feels peaceful, I’ll stay." (Joy Joy, 312 likes)  
   "‘I should be doing this’ ... weight in my stomach ... rephrased ‘I want to finish this because it matters to me, because I choose to’. The weight lifted." (Su, 282 likes)  
   → obligation/streak pressure = 이탈, choice + peaceful + grace = 지속.

- **싫어하는 것**: streak가 목적화 (deep thinking 희생), "should / forced / obligation", cloud lock-in / data not mine 느낌 (creepy / surveillance 우려), guilt-driven daily nag.
- **KR 특이**: restart tolerance 높음 (작심삼일 문화 수용), 로컬 sovereignty ("폰에만 저장... 프라이버지 걱정도 없고")가 entry/trust precondition으로 강하게 작동. Kakao/Apple optional later, first value 즉시 제공 시 no-account 허용 신호.
- **기회 (T1 P1 + Lever B + week-1)**: first journal save gate 해제 순간 + pre-account capture + re-entry에 grace + recap + local banner를 hard immediate로 넣으면 time-to-aha <30s + retention glue + acquisition wedge (Obsidian 스타일). "작심이틀" copy로 KR fit 극대화.
- **리스크**: streak culture residual (US competitor 영향) or "recap"을 또 다른 pressure로 느끼게 만들면 역효과. Privacy banner 없이 local만 강조하면 shallow trust.
