## 기회·리스크
**기회 (ownership/portability 강점)**:  
- "your records, your ownership, portable/export" 메시지가 Obsidian·vault·local-first 포스팅에서 일관·고engagement 호응. 2nd-B의 "second brain" 정체성과 완벽 매치. "No lock-in", "plain files", "export everything", "you control"가 사용자 신뢰를 빠르게 쌓는 핵심 언어로 확인.  
- "AI remembers / gets to know you" 는 **local/on-device + user-directed + opt-in + visible** 조건 하에서 "living journal", "being seen", continuity로 긍정 전환 가능 (Harbour, poetry witness 사례).

**리스크 (creepy "learns you")**:  
- 불투명한 "quietly built a file on you", "database entries", "profiling without consent"에 강한 거부. levelsio급 고영향력 계정에서 "sociopath" 비유까지 나옴. Thread에서 "invited vs assumed", "consent"가 핵심 분기점.  
- "AI friend" 카피는 관계성(being seen) 기대를 높이지만, journaling/self-recording 사용자 중 privacy-conscious layer(특히 notes=unpolished truths 공유 꺼림)에서 backlash 유발 가능. (signulll thread + levelsio 교차 확인)

상충 신호: **단정 금지**. 같은 "AI deeply understands personal data"를 spiritual/positive로 보는 집단 vs surveillance/creepy로 보는 집단 병존. Engagement 높은 쪽은 ownership 투명성과 consent 명시 시 우호적.
