## X/Consumer Signals (Fresh Synthesis, 2026-06)
Key patterns from recent X consumer/PM discourse on journaling, second-brain, PKM, habit apps (global + KR-adjacent habit signals):

- **Onboarding contract fairness (Hiten Shah core)**: Users silently churn when early asks (email/perm/spinner) exceed immediate reward. "Every screen is a contract." TikTok/Duolingo model: swipe/tap → instant payoff → next feels fair. Financial/PKM apps show D1 ~28%, D7 ~17-18%, D30 ~8%. First capture must deliver visible tiny win before any gate.
- **Restart grace > motivation myth (Idan Refaeli)**: Churn after 3-day gap often from "disappointed parent" UX (broken streaks, guilt dashboards, yelling notifs). Winners make return feel normal/clean: no shame, instant "resume last note" path. One indie: added "resume from last note" → D7 retention 31% → 47% in 3 weeks.
- **Aha moment speed + effortless capture (multiple founders)**: Users come back for dopamine/tiny wins/instant feedback, not abstract "usefulness". PKM specifically dies on capture friction: "if saving a thought takes any friction people stop and the brain goes empty." Browser/share/quick-add > fancy features. Proactive recall surfaces value later.
- **Week-1 quick win + momentum before doubt (Alex Wells, jack friks)**: First 14 days set temp. Visible quick win (connect+create in one flow) inside week 1 beats long onboarding. Checklist to AHA, not questions. One case: removed pre-AHA friction → better trial start + first-week activation (churn still high at 22% but directionally better).
- **Ownership & cognitive permanence (user voices)**: Hand-writing advocates cite "cognitive benefit of fully owning the code line by line" + "knowledge retention/gain". Second-brain narratives sell compound edge (Week 1: clarity; Month 1: projects move; Month 6: your edge). Local/permanent ownership is sticky hook vs cloud nag.
- **KR/habit signals (limited direct app reviews; general patterns)**: Habit formation posts emphasize consistency without future-sacrifice guilt ("마시멜로 미래 미루다 유통기한 지나"). First-touch opt-in, low-pressure entry, graceful re-entry align with KR preference for non-intrusive starts (Kakao/Apple social low barrier noted in prior).

4. **Week-1 Quick-Win + Ownership/Permanence Lens** (AHA timing + cognitive own): Users need visible momentum and sense of "I own this" fast. Local-first pending + graceful first-write permanence signals (no cloud lock-in vibe) beat feature lists. "Resume last" is high-ROI for D7.

Verdict synthesis: Consistent across lenses — low-pressure + instant ownership + grace = week-1 wedge. High-pressure or delayed value = predictable churn per public PM benchmarks.
