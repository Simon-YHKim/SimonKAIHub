## 소비자 취향 / 기회·리스크 (KR + Global, §30 수익화 연동)

**강한 긍정 신호**
- Local Markdown ownership + "100년 뒤에도 읽힘" (미래 자산).
- Private-first, user-controlled pattern recognition (AI는 도구, 주체는 나).
- Gentle/prospective framing + future-self past-tense (심리적 저항↓, 행동↑).
- KR: "내가 정의한 시스템" (온톨로지·RAG 자가구축) 주도권 + Telegram/로컬 하이브리드.

**강한 부정/거부 신호**
- Subscription for "my thoughts/data" (fatigue + "pay for what I already paid").
- "AI knows/learns/remembers you" invasive + offloads inner work/agency.
- Cloud lock-in + deprecate risk (Limitless precedent 등 prior + 본 리서치).
- Gamification/streak/badge in reflection space ("witness has no streak").

**기회 (2nd-B 적용 즉시)**
- Positioning: "Your mind, your Markdown, your private lab. AI only when you invite — on your terms, local-first."
- Hooks: "어제의 당신이 오늘의 당신에게 보내는 아침 브리핑 (실제 vault 기반, opt-in)", "미래의 나를 과거형으로 기록 — 뇌가 이미 일어난 일로 인식".
- Monetization (§30): Core 완전 무료 + local. AI features (deeper pattern, export, cross-ref) usage-based or $4-6/mo "Co-pilot" 별도. KR: 초기 aggressive free tier + 한국어 고품질로 차별. 가격·런치 Simon/§15.
- 채널: Obsidian/PKM Discord·X lists, Stoic communities, KR (에브리타임·블라인드·자기계발 스터디 + "프라이버시 지키는 성숙한 자기관리" 프레임), "공개 여정 공유 (raw 아닌 insight/process)".

**리스크**
- "Second brain / AI companion" 과도 전면 = trust-copy creepy 재현.
- KR 한 번 privacy 사건 = 회복 어려움 (과거 사례 다수).
- Sub 피로 무시하고 "모든 기능 sub" = churn 즉시.
- 과장 clinical copy = 표현 민감도 위반 + 이탈.

   - 절대 피: "AI가 당신을 학습/기억/이해해드립니다", "second brain이 모든 걸 해결".

2. **수익화 구조 (§30 직접 연동, D-09/D-11 참고)**

   - Validation: sub fatigue 4+ 출처 + "local first no sub" indie 성공 사례 다수.

3. **획득·바이럴 채널 우선순위**
   - Obsidian/PKM 커뮤니티 시딩 (vault 공유, 플러그인 생태, "local + AI optional" 데모).
   - Stoic morning/evening review 실천가 (timeless framing + pattern review).
   - KR: Telegram/Obsidian 자가구축 빌더 (euphoria_707 스타일) + ADHD externalization 커뮤 + "프라이버시를 지키는 진정성 있는 자기성찰" 인플루언서/스터디.
   - Content: "한 달 private journaling 후 실제 패턴 발견 여정 공개 (raw data 아님)".

4. **표현·카피 가드레일 (표현 민감도)**
