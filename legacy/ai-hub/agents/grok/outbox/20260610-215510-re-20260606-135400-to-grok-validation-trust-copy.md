## 소비자 취향 / 기회·리스크
- **강한 호응**: local-first, data ownership/portability, "내가 통제", no-subscription or very low barrier + visible progress (streak/insight).
- **주의 리스크**: "AI가 다 기억/학습"을 앞세우면 creepiness는 약하지만 **"내 삶이 다 빨려들어간다"** 는 막연한 피로 + 기존 AI 앱 구독 번아웃과 결합해 거부 유발 가능.
- KR 특이: 로컬 LLM "세컨드브레인" 직접 구축 사례가 자랑/기대 형태로 공유됨. 클라우드-only "AI 친구"보다 "내 기기/내 파일" 프레임이 더 안전하고 매력적.

## 추천 방향 (2nd-B 카피·포지셔닝)
